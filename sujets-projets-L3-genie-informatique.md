## Liste des thèmes de recherche (30 sujets)

Voici les thèmes proposés pour les projets de fin d'études. Ils sont classés par domaine pour faciliter votre choix.

### I. Gestion Administrative et Sécurité

**1. Certification numérique des diplômes**
- **Objectif** : Vérifier l'authenticité des diplômes via une empreinte unique.
- **Stack** : Python (Flask), React.js, SHA-256, PostgreSQL.

**2. Plateforme de signalement anonyme**
- **Objectif** : Recueillir les dysfonctionnements sans identifier les auteurs.
- **Stack** : PHP (Laravel), Tailwind CSS, PostgreSQL.

**3. Vote électronique sécurisé**
- **Objectif** : Moderniser les élections des délégués.
- **Stack** : Python (Django), Chiffrement RSA/AES, PostgreSQL.

**4. Archivage numérique des dossiers administratifs**
- **Objectif** : Numériser et indexer les archives pour une recherche rapide.
- **Stack** : Node.js (Express), Vue.js, Tesseract OCR, MySQL.

**5. Gestion sécurisée des bourses d'études**
- **Objectif** : Automatiser le suivi des paiements et des dossiers de bourse.
- **Stack** : Java (Spring Boot), React, PostgreSQL.

**6. Système de gestion des badges d'accès par NFC**
- **Objectif** : Contrôler l'accès aux salles sensibles (serveurs, labos).
- **Stack** : Arduino/ESP32, Python, MySQL, Application Android.

### II. Suivi Pédagogique et Vie Étudiante

**7. Pointage automatique des étudiants par QR Code**
- **Objectif** : Suivi des présences en temps réel via application mobile.
- **Stack** : Flutter, Node.js, QR Codes dynamiques, MySQL.

**8. Gestion automatisée des heures d'enseignement**
- **Objectif** : Calculer et valider les heures des enseignants.
- **Stack** : PHP (Laravel), React, DomPDF.

**9. Portail de gestion des stages et mémoires**
- **Objectif** : Suivre le parcours des étudiants de la recherche de stage à la soutenance.
- **Stack** : Python (Django), Vue.js, MySQL.

**10. Réseau social des Alumni (Anciens étudiants)**
- **Objectif** : Maintenir le lien entre les diplômés et l'université.
- **Stack** : Next.js, Firebase (Auth & Firestore), Tailwind CSS.

**11. Système de réservation de salles et de matériels**
- **Objectif** : Éviter les conflits d'occupation des salles de cours.
- **Stack** : Node.js, React, WebSockets (temps réel), MySQL.

**12. Portfolio numérique et générateur de CV pour étudiants**
- **Objectif** : Aider les étudiants à valoriser leurs compétences académiques.
- **Stack** : React, Node.js, Puppeteer (génération PDF).

### III. Intelligence Artificielle et Data Science

**13. Assistant de recommandation de cours (IA)**
- **Objectif** : Suggérer des ressources selon le profil de l'étudiant.
- **Stack** : Python, Scikit-learn, FastAPI.

**14. Détection de textes générés par IA (ChatGPT)**
- **Objectif** : Vérifier l'originalité des travaux remis.
- **Stack** : Python, NLP (HuggingFace, Spacy), Flask.

**15. Chatbot d'assistance administrative**
- **Objectif** : Répondre aux questions fréquentes sur WhatsApp/Web.
- **Stack** : Rasa ou Dialogflow, Node.js, API WhatsApp Business.

**16. Analyse de l'efficacité des processus par Process Mining**
- **Objectif** : Identifier les goulots d'étranglement administratifs.
- **Stack** : Python, PM4Py, Matplotlib.

**17. Tableau de bord de pilotage pédagogique (BI)**
- **Objectif** : Visualiser les taux de réussite et les moyennes.
- **Stack** : Python (Pandas), D3.js ou Metabase, PostgreSQL.

**18. Prédiction du décrochage étudiant par Machine Learning**
- **Objectif** : Identifier les étudiants en difficulté dès le début du semestre.
- **Stack** : Python, Pandas, Scikit-learn (Random Forest/SVM).

**19. Analyse de sentiments sur les feedbacks étudiants**
- **Objectif** : Évaluer la satisfaction des étudiants sur les cours.
- **Stack** : Python, NLP (TextBlob ou VADER), Streamlit.

**20. Générateur automatique d'emplois du temps**
- **Objectif** : Optimiser l'usage des salles et le temps des professeurs.
- **Stack** : Python (Algorithmes génétiques), Flask.

### IV. Services Numériques et IoT

**21. Gestion de bibliothèque avec code-barres/RFID**
- **Objectif** : Automatiser les prêts et les retours de livres.
- **Stack** : Python (PyQt), Lecteur RFID, MySQL.

**22. Application de navigation sur le campus (Plan interactif)**
- **Objectif** : Guider les nouveaux étudiants vers les différents services.
- **Stack** : Flutter, Mapbox API, OpenStreetMap.

**23. Plateforme de cours en ligne (LMS léger)**
- **Objectif** : Partage de supports de cours et quiz en interne.
- **Stack** : PHP (Laravel), Vue.js, MySQL.

**24. Système de paiement numérique pour la cantine (Wallet)**
- **Objectif** : Supprimer le cash via des paiements par QR Code.
- **Stack** : Flutter, Node.js (Express), MongoDB.

**25. Gestion intelligente de l'énergie (Éclairage des salles)**
- **Objectif** : Économiser l'électricité via des capteurs connectés.
- **Stack** : ESP32 (MicroPython), MQTT, Tableau de bord (Grafana).

**26. Plateforme de concours d'entrée en ligne**
- **Objectif** : Digitaliser les tests de sélection des nouveaux étudiants.
- **Stack** : Angular, Node.js, Socket.io (pour la surveillance).

**27. Dossier médical numérique de l'étudiant**
- **Objectif** : Centraliser le suivi de santé au sein de l'université.
- **Stack** : PHP (Laravel), Chiffrement des données, MySQL.

**28. Système de stockage Cloud pour les travaux de recherche**
- **Objectif** : Permettre aux étudiants de sauvegarder leurs projets.
- **Stack** : Node.js, MinIO (Stockage objet), React.js.

**29. Outil de comparaison de documents (Anti-plagiat interne)**
- **Objectif** : Détecter les copies entre étudiants d'une même promo.
- **Stack** : Python (Cosine Similarity), Flask.

**30. Plateforme de "Journées Portes Ouvertes" virtuelles**
- **Objectif** : Présenter les filières aux futurs bacheliers à distance.
- **Stack** : Three.js (3D web), React.js, Firebase.
